# Text Elements
example:
class ... { .. float x; ...}

instead of
number x;

don't need details about "x", only the methods it supports ^dNmV5QUi

example:
class Cname { ... }

instead of
Thing Cname;

don't need to know "how" Cname is constructed, only need to know that Cname exists ^4TxoJ7Bi

Too Much Detail -> Bloatware ^TByXE98t
